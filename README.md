# THESIS

Thesis python pipeline for identifying critical positions in chess

Stockfish.py collects and preprocess data using Stockfish

Analysis.py implements and analyses different strategies and position pools they gather

Datasets of both pipelines will be shared in this branch

Original source dataset:
Chess.com Titled Tuesday Games (Stassl, 2024)
https://www.kaggle.com/datasets/stassl/chess-com-titled-tuesday-games
